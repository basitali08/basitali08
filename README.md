## 📌 Latest Projects

### 🧬 [Neuron Receptor Binding Prediction](https://github.com/basitali08/neuron-receptor-prediction)
Multi-class ML model predicting which brain receptor a drug binds to (Dopamine D2, 5-HT2A, GABA-A, NMDA, Mu-Opioid). **60% accuracy, 0.84 ROC-AUC** across 5 classes. Published on SSRN.
[![SSRN](https://img.shields.io/badge/Paper-SSRN-blue)](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6852319)

### 🚗 [Vehicle Tracking & Hotlist Alert System](https://github.com/basitali08/vehicle-tracking-system)
Real-time ANPR system: detects license plates from images using EasyOCR, matches against a hotlist database, and plots detections on an interactive Folium map with color-coded alerts.

### 🧠 [Schizophrenia Drug Response Predictor](https://github.com/basitali08/schizophrenia-drug-predictor)
Predicts the best antipsychotic drug (out of 5) from 15 patient features. **Random Forest: 74% accuracy, 0.92 AUC** — 3.7x better than random. Helps doctors skip trial-and-error.
