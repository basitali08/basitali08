## 📰 Featured Publication

### 🧬 [Machine Learning Models for Predicting Neuron Receptor Binding](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6852319)
**Ali, B.** (2025). *SSRN Electronic Journal*. DOI: [10.2139/ssrn.6852319](https://doi.org/10.2139/ssrn.6852319)

A benchmark study evaluating multi-class classification and regression approaches for predicting drug binding affinity across **5 neuron receptors**:
- **Dopamine D2** · **Serotonin 5-HT2A** · **GABA-A** · **NMDA** · **Mu-Opioid**
- **47.7%** multi-class accuracy (XGBoost) across 5 classes
- **76.4%** binary accuracy for Serotonin 5-HT2A
- **R² = 0.49** for pIC50 regression
- 8 engineered molecular features · SMOTE-balanced
